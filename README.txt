Version 1.0
* Removed the maintain sound. It still "plays" but the sound file is blank.
* Added Neimans item to distribution tables.
* Added .223 to 5.56 recipe. Some occupations start with the recipe learned and can make notes to share it.
* Added "Corvus' Cookbook" which contains all the random recipes that dont fit anywhere else.
* Added a more time efficient vinegar recipe. With an increased skill requirement and resource cost that is friendlier with extended day length.
* Modified armour values of MILITARY PONCHOS from the Military Ponchos Expanded Mod. 
* Herbal tea can be made in a wider variety of mugs. Now you can have tea in a spiffo mug!
* BEANS! Can now be used in most recipes that would take tinned beans.
* Reduced spawn rates of 4.6 and 5.7 by 50% to account for limited weapons which utilise it. (Testing.)
* Added recipe to create a military poncho from a yellow tarp then recipes to paint it various colours.
